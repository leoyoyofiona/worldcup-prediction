# 2026 男足世界杯预测小网页

一个本地运行的小型预测网页。它不需要用户提供 API Key 或数据，后端会从公开数据源抓取赛程、历史国家队比赛结果和市场热度页面，构建可解释的小模型，并在网页里展示每场比赛的预测。

页面同时展示完整赛事推演：32 强、16 强、8 强、4 强/半决赛、决赛和冠军概率。

## 运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

打开 `http://127.0.0.1:8000`。

首次打开时，如果本地没有缓存，页面会自动触发一次数据更新。也可以点击页面右上角的“更新数据”手动刷新。

## 模型

模型是可解释规则模型，不使用付费数据或隐藏 API Key。当前版本参考高盛公开世界杯预测思路，把长期实力、正式比赛攻防、世界杯历史、主办国/主大陆和市场信号组合后进行大规模模拟：

- 历史国际比赛结果计算 Elo。
- 最近 10 场计算近期状态，最近 20 场估计攻防强度。
- 参考高盛公开模型，额外加入正式比赛近 10 场进攻/失球、近 5 场正式比赛防守、世界杯正赛历史表现、世界杯阶段/淘汰赛经验、主大陆因素。
- Poisson 进球分布生成比分矩阵和胜/平/负概率。
- 美国、墨西哥、加拿大作为东道主获得小幅主办国修正。
- 义乌/外贸/世界杯周边订单相关页面只作为市场热度辅助信号，最多带来约 25 Elo 的相对修正。
- 公开博彩盘口页面只作为赔率市场辅助信号，最多带来约 35 Elo 的相对修正；不构成投注建议。
- 淘汰赛推演使用固定随机种子进行 50,000 次模拟，并输出阶段概率和预测对阵。

## 数据源

默认公开来源包括：

- openfootball 的 2026 World Cup JSON 赛程。
- martj42/international_results 的国际比赛历史 CSV。
- 本地 `WorldCupMatches.csv` 历届世界杯正赛数据。程序会优先读取项目根目录或 `data/WorldCupMatches.csv`，也会读取 `/Users/leo/Downloads/世界杯数据/WorldCupMatches.csv`。
- FIFA 男足世界排名页面。
- 小商品指数网、商务预报、中国商品网、DuckDuckGo 公开搜索结果页中的义乌世界杯订单相关文本。
- OddsJet 多地区页面和 Compare.bet 世界杯冠军赔率页，用于公开盘口市场信号。

抓取结果、模型输出和来源状态会缓存在 `data/` 目录。该目录不会提交到 Git。
